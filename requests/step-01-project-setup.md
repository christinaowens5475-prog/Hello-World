# Step 1 — Project Setup

## Task
Scaffold a new Next.js 15 project with TypeScript and Tailwind CSS, then configure the environment for the Long Beach weather app.

## Instructions
1. Run the Next.js create command with TypeScript, Tailwind CSS, App Router, and no src/ directory
2. Install any additional dependencies needed
3. Create a `.env.local` file with a placeholder for the OpenWeatherMap API key
4. Verify `.env.local` is listed in `.gitignore`
5. Remove the default Next.js boilerplate from `app/page.tsx` and `app/globals.css`

## Acceptance Criteria
- Project runs with `npm run dev` without errors
- `.env.local` exists with `OPENWEATHERMAP_API_KEY=` placeholder
- `.gitignore` includes `.env.local`
- No default boilerplate content remains
